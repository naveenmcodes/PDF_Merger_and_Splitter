Project Name: PDF Merger & Splitter

Scope: A Python-based utility that allows users to merge multiple PDF files into a single document and split PDF files into individual pages.

Features:

Merge multiple PDF files into one PDF

Split a PDF into separate pages

Automatic creation of output folders

Simple menu-driven interface

Fast and easy PDF processing

Technologies Used:

Python
pypdf Library
os Module
datetime Module
Installation

Install the required library:
pip install pypdf

How to Run:

Open the project in PyCharm or any Python IDE.
Run the Python script.

Select an option:
Merge PDFs
Split PDF
Exit
Enter the required file paths.
The processed PDF files will be generated automatically.

Project Outcome:

The application simplifies PDF management by allowing users to merge documents and split PDFs into individual pages efficiently using Python.
